## Installation
### Cargo
- Install the rust toolchain in order to have cargo installed by following [this](https://www.rust-lang.org/tools/install) guide.
- run `cargo install shuttle-christmas-2023`

## shuttle verify:
```
cargo shuttle run
cargo install cch23-validator
cch23-validator -1
```
